**Customer Behavior Analysis Using Python, MySQL, and Power BI**

Developed a **Customer Behavior Analysis** project to analyze customer purchasing patterns, identify trends, and generate actionable business insights. The project involved extracting and managing customer transaction data using **MySQL**, performing data cleaning, preprocessing, and exploratory data analysis (EDA) with **Python** (Pandas), and creating interactive dashboards in **Power BI**.

Key features included customer segmentation, purchase frequency analysis, sales trend visualization, top-selling product identification, and revenue analysis. SQL queries were used to retrieve and manipulate large datasets efficiently, while Power BI dashboards provided dynamic visualizations with filters and KPIs to support data-driven decision-making.

**Technologies Used:** Anaconda, Jupyter , Python, MySQL, Power BI

**Key Outcomes:**

* Cleaned and transformed raw customer data for analysis.
* Performed exploratory data analysis to uncover customer purchasing behavior.
* Identified high-value customers and buying patterns through segmentation.
* Built interactive Power BI dashboards to visualize sales performance, customer demographics, and business KPIs.
* Improved business decision-making by presenting clear, data-driven insights through reports and visualizations.
